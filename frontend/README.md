# FindPhotos

Expo Router app for Organizer/Attendee photo workflow with Firebase email/password auth and selfie capture.

## Setup

1. Copy `.env.example` to `.env`.
2. Fill in Firebase web app values.
3. Install dependencies:

```bash
npm install
```

4. Start Expo:

```bash
npm run start
```

## Tech

- Expo Router navigation
- Firebase Auth (email/password)
- `expo-camera` selfie capture
- Dark UI theme with accent `#6C63FF`
